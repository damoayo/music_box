# Next로 Music Player 사이트 만들기

포트폴리오 컨셉의 사이트를 4가지 버전으로 반복하여 만들어 보면, 웹사이트의 일련의 제작과정을
이해할 수 있고, 연습을 통해서 제작기법을 숙지할수 있고 다양한 방법을 사용하여
흐름과 감각을 익히기 좋은 방법이라고 생각합니다.
